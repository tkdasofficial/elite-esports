BUILD A COMPLETE "AD TAG MANAGEMENT SYSTEM" PAGE FOR A GLOBAL ESPORTS PLATFORM NAMED "ELITE ESPORTS"

GOAL:
Create a fully functional system where ADMIN can add, manage, and control advertisement tags (AdMob, AdSense, custom scripts, links) dynamically from backend WITHOUT modifying the app codebase. This system must work for Web + React Native App using API-based dynamic rendering.

🔷 BACKEND (TYPECRIPT + API DESIGN)

TECH STACK:

- Node.js + Express (or Next.js API)
- TypeScript
- Database: PostgreSQL / Supabase

DATABASE TABLE: ad_tags

FIELDS:

- id (uuid)
- name (string)
- type (enum: admob, adsense, custom, redirect)
- platform (enum: web, android, ios, all)
- placement (string)
- code (text)  // script or ad unit id
- trigger_type (enum: load, click, timed)
- delay (integer)
- priority (integer)
- frequency_limit (string)
- status (boolean)
- created_at
- updated_at

---

🔷 API ENDPOINTS

POST /api/ad-tags
→ Create new tag

GET /api/ad-tags
→ Get all tags

GET /api/ad-tags?placement=home_banner
→ Filter tags by placement

PUT /api/ad-tags/:id
→ Update tag

DELETE /api/ad-tags/:id
→ Delete tag

GET /api/ad-tags/active?placement=reward_button&platform=android
→ Fetch only active tags for runtime usage

---

🔷 CORE SYSTEM LOGIC

IMPORTANT:

- Ads MUST NOT be hardcoded in app
- App will FETCH ads dynamically from backend

FLOW:

1. App opens
2. App calls API:
   GET /api/ad-tags/active
3. Backend returns only ACTIVE tags based on:
   - platform
   - placement
4. App decides WHEN to show ad based on:
   - trigger_type
   - delay
5. If user clicks button → trigger ad
6. If timed → run after delay

---

🔷 REACT NATIVE IMPLEMENTATION LOGIC

CREATE A FUNCTION:

function loadAds(placement) {
call API → /api/ad-tags/active?placement=placement
store ads in state
}

DISPLAY LOGIC:

IF type = admob:
→ use react-native-google-mobile-ads SDK
→ load using Ad Unit ID

IF type = adsense:
→ only for WebView (not direct RN support)

IF type = custom script:
→ render inside WebView

IF type = redirect:
→ open external link

---

🔷 IMPORTANT LIMITATIONS (VERY IMPORTANT)

- AdMob CANNOT run from raw script → needs SDK integration
- AdSense does NOT work natively in React Native → only via WebView
- You CANNOT inject arbitrary JS into native app runtime (security restriction)
- So:
  ✔ Backend controls WHAT ad to show
  ✔ App controls HOW to render it using pre-integrated SDK
